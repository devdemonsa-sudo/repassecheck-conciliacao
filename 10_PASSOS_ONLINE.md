## 🆘 Se Algo Dar Errado

### Erro 404 no Vercel
→ Verifique se todos os arquivos foram para o GitHub  
→ Acesse: vercel.com → seu projeto → "Deployments" → veja os logs

### Página em branco
→ Limpe o cache: Ctrl+Shift+Delete  
→ Recarregue com Ctrl+Shift+R (hard refresh)  
→ Aguarde 1 minuto

### Estilos não aparecem
→ Tente acessar em aberta anônima  
→ Ou acesse: site.vercel.app/?cache=false
